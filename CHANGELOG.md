# 변경 기록

## V0.0.5
- 좌측 워크플로우 영역을 더 얇은 세로 탭 바 형태로 다시 조정했습니다.
- 단계 탭을 세로 스택으로 배치하고, 선택된 탭 상세가 오른쪽 패널에 보이도록 구조를 더 분명하게 맞췄습니다.

## V0.0.4
- Dribbble의 Trior 대시보드 방향을 참고해 홈 화면과 상세 화면을 대시보드형 UI로 재구성했습니다.
- 좌측 고정 사이드바, 상단 요약 영역, 카드형 입력 패널, 상태 카드 구성을 적용했습니다.
- 홈 화면에서도 워크플로우 진행 구조가 보이도록 정보 배치를 다시 정리했습니다.

## V0.0.3
- 현재 작업 폴더를 `REPO` Git 저장소에 연결했습니다.
- 앞으로 버전업 기준으로 이 저장소를 사용하도록 맞췄습니다.
- 현재는 원격 저장소 URL이 없어서 GitHub 푸시는 불가한 상태입니다.

## V0.0.2
- 버전 규칙을 `major.minor.patch`로 고정했습니다.
- 루트 변경기록 파일을 추가했습니다.
- 실행 주소 안내를 `http://127.0.0.1:3100` 기준으로 수정했습니다.
- 서버 창을 닫으면 접속이 끊기는 점을 문서에 명시했습니다.
- 현재 GitHub 계정 인증은 확인됐지만 원격 저장소 연결은 없어 푸시는 보류 상태입니다.

## V0.0.1
- 롱폼 자동 공장 기본 프로젝트를 생성했습니다.
- 주제, 리서치, 대본, 장면, 렌더, 업로드 흐름의 MVP를 추가했습니다.
- n8n 연동 예시와 영상 렌더링 파이프라인을 추가했습니다.
