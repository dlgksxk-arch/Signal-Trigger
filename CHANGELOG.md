# 변경 기록

## V1.0.33
- 이미지 프롬프트를 Draph 가이드 방식에 맞춰 `주제 -> 스타일 -> 구도/환경 -> 세부사항` 구조로 정리했습니다.
- 예시 이미지 느낌에 맞게 `평면 벡터 일러스트`, `굵은 외곽선`, `밝은 파스텔`, `간결한 배경`, `읽기 쉬운 구도`를 기본 스타일로 고정했습니다.

## V1.0.32
- 리서치 자동 수집 로직에 `Signal Trigger`용 스토리 스카우트 프롬프트를 연결했습니다.
- 트렌드를 그대로 저장하지 않고 클릭감 있는 15개 주제 후보와 훅 각도를 리서치 결과로 저장하도록 바꿨습니다.

## V1.0.31
- 주제 프롬프트 언어와 상관없이 프로젝트 설정에서 선택한 언어를 기준으로 주제와 대본을 생성하도록 수정했습니다.

## V1.0.30
- 이미지 생성 프롬프트를 더 비비드한 팝아트 스타일로 조정했습니다.

## V1.0.29
- 초기화 요청 시 진행 중인 더빙과 렌더까지 중단되도록 수정했습니다.
- 초기화 직후 화면 상태와 진행 정보가 즉시 비워지도록 보정했습니다.

## V1.0.28
- 20~30대 타겟에 맞춰 대본과 이미지 톤을 더 빠르고 쉽게 따라가는 방향으로 조정했습니다.

## V1.0.27
- 이미지 생성 기본 프롬프트를 더 강한 팝 임팩트 스타일로 교체했습니다.

## V1.0.26
- `Signal Trigger`용 장문 스토리텔링 대본 프롬프트를 기본값으로 반영했습니다.

## V1.0.25
- 자막 전환 시점을 장면 전환 시점과 맞추도록 수정했습니다.

## V1.0.24
- 장면 수를 영상 길이 기준으로 계산하도록 바꾸고 `6초당 1장` 규칙을 적용했습니다.

## V1.0.23
- 프로젝트 제목을 생성/수정할 수 있게 추가했습니다.

## V1.0.22
- 프로젝트 상세에서 스타일 레퍼런스, BGM, 워터마크 파일을 다시 업로드하거나 삭제할 수 있게 했습니다.

## V1.0.21
- OpenAI 이미지 생성 기본 품질을 `low`로 낮췄습니다.

## V1.0.20
- 이미지 기본 모델을 `gpt-image-1`로 변경했습니다.
- 장면 이미지에 줌/패닝 애니메이션 느낌을 넣고 썸네일 문구를 더 짧고 강하게 정리했습니다.

## V1.0.19
- 버전 파일 간 불일치를 정리하고 감시 모드 실행 기준으로 맞췄습니다.

## V1.0.18
- `npm run start`가 감시 모드로 동작하도록 바꿔 코드 수정 후 버전 반영이 더 잘 되게 정리했습니다.

## V1.0.17
- 대시보드에서 `noticeMessage`가 없을 때 발생하던 렌더 오류를 막았습니다.

## V1.0.16
- 프로젝트 삭제 기능을 추가하고 생성된 출력 폴더까지 함께 삭제되도록 구현했습니다.
